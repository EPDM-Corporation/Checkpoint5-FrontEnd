# Projeto Melodia
Este projeto teve como o objetivo o desenvolvimento web baseado em sites de streaming de músicas, utilizando ferramentas como React + Vite e Tailwind.

## Funcionalidades
- Carrosel de imagens.
- Design responsivo.
- Header fixo com efeito de transparência.
